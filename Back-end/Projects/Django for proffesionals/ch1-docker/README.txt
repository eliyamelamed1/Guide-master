Conclusion

• create a virtual environment locally and install Django
• create a new project
• exit the virtual environment
• write a Dockerfile and then build the initial image using : $ docker build .
• write a docker-compose.yml file and run the container using: $ docker-compose up
• to confirm it worked go to http://127.0.0.1:8000/
• close server with :  ctrl + c THEN $ docker-compose down
